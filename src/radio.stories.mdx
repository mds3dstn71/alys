# Radio

Forms normally have sections where the user is required to select one and only one choice from a
list of choices. When the list of choices is short, a set of radio inputs may be enough.

## Demo

Select a song for your ringtone. You can preview the song by clicking on the audio player's play
button. You can set the player to instantly play when the song changes by ticking the "Auto play"
checkbox in the Options section.
