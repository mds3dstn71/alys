# Tag

Tags can be used to show how a certain item is categorized. It appears a lot on collections, below
the title tag to indicate that the current item belongs to a group. When clicked, it usually
navigates to a separate page to show all the items tagged with a similar tag.

## Overview

Solid tags are normally used to draw more attention to function as badges to indicate short but
important information, e.g. deprecation notices, new features, etc.

Secondary tags will also provide information but it is not as critical. Since it's not eye-catching
as the solid tags, they would not detract attention from more important UI elements, i.e. when
showing a list of cards, we would normally want to make the card titles or card descriptions to be
more prominent than the card tags.
