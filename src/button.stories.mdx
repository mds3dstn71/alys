# Button

Buttons are used to for providing the user a way to communicate that they want an action to be
performed. Examples of action are confirmation of a process, issuing a request to fetch some data,
etc.

## Overview

Here are all the primary buttons. They all have a solid background color.

### Variant (primary)

Solid buttons are normally used to draw more attention. They are most likely used for call-to-action
(CTA) buttons, confirmations, etc.

<Preview>
  <Story name='Primary variant'>
    <Button onClick={action('clicked')}>
      Primary
    </Button>
  </Story>
</Preview>

### Variant (secondary)

Secondary buttons maybe intended for some optional action like, canceling or going back to a
previous page.

### ReactNode as children (default size)

Adding icons to button is done by importing the icon and placing it along the button text. You
should put appropriate margins so that the icon and the button text are not stuck together.

<Preview>
  <Story name="Default sized button with an icon">

      Download

    </Button>
