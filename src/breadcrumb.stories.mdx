# Breadcrumb

Breadcrumbs are navigation components that allow a user to see the parents and grandparents of the
page they are currently viewing.

## Usage

Wrap a list of BreadcrumbItems inside a Breadcrumb component. The BreadcrumbItem will accept the
path (`href`) and anchor text (as child). By default, any last BreadcrumbItem will be marked as
`active`.

<Preview>
  <Story name="Breadcrumb/Example">
    <Breadcrumb>
      <BreadcrumbItem href="/">Alys</BreadcrumbItem>
      <BreadcrumbItem href="/components">Components</BreadcrumbItem>
      <BreadcrumbItem href="/components/breadcrumb">Breadcrumb</BreadcrumbItem>
      <BreadcrumbItem href="/components/breadcrumb/breadcrumb-item">BreadcrumbItem</BreadcrumbItem>
    </Breadcrumb>
  </Story>
</Preview>

## BreadcrumbItem

Breadcrumb items is provided the link text, path and if that item is the currently active page.

<Preview>
  <Story name="BreadcrumbItem/Active/False">
    <BreadcrumbItem href="/components">Components</BreadcrumbItem>
  </Story>
</Preview>

Although marking a BreadcrumbItem as `active` is not required, it is still possible to provide a
boolean `active` attribute to a BreadcrumbItem but this will be ignored if the BreadcrumbItem is a
child to a Breadcrumb wrapper component.

<Preview>
  <Story name="Breadcrumb/Item/Active/True">
    <BreadcrumbItem active href="/components/breadcrumb/breadcrumb-item">BreadcrumbItem</BreadcrumbItem>
  </Story>
</Preview>
