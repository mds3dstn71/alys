# Breadcrumb

Breadcrumbs are navigation components that allow a user to see the parents and grandparents of the
page they are currently viewing.

## Usage

This next example is lifted straight out of the WAI-ARIA Authoring Practices breadcrumb example.

<Story name="Basic/ARIAExample">
  <div style={{ width: '100%', display: 'flex', justifyContent: 'space-around', }}>
    <Breadcrumb links={[
        {
          href: '../..',
          text: 'WAI-ARIA Authoring Practices 1.1',
        },
        {
          href: '../../#aria_ex',
          text: 'Design Patterns',
        },
        {
          href: '../../#breadcrumb',
          text: 'Breadcrumb Pattern',
        },
        {
          href: './index.html',
          text: 'Breadcrumb Example',
          isCurrentPage: true,
        },]} />
  </div>
</Story>
