# Breadcrumb

Breadcrumbs are navigation components that allow a user to see the parents and grandparents of the
page they are currently viewing.

## Usage

Wrap a list of BreadcrumbItems inside a Breadcrumb component.

<Preview>
  <Story name="Demo">
    <Breadcrumb>
      <BreadcrumbItem href="#">Alys</BreadcrumbItem>
      <BreadcrumbItem href="#">Components</BreadcrumbItem>
      <BreadcrumbItem href="#">Breadcrumb</BreadcrumbItem>
      <BreadcrumbItem href="#">BreadcrumbItem</BreadcrumbItem>
    </Breadcrumb>
  </Story>
</Preview>
