# Breadcrumb

Breadcrumbs are navigation components that allow a user to see the parents and grandparents of the
page they are currently viewing.

## Usage

This next example is lifted straight out of the WAI-ARIA Authoring Practices breadcrumb example.

### Basic ARIA example

<Preview>
  <Story name="Breadcrumb/ARIAExample">
    <Breadcrumb
      links={[
        {
          href: '../..',
          text: 'WAI-ARIA Authoring Practices 1.1',
        },
        {
          href: '../../#aria_ex',
          text: 'Design Patterns',
        },
        {
          href: '../../#breadcrumb',
          text: 'Breadcrumb Pattern',
        },
        {
          href: './index.html',
          text: 'Breadcrumb Example',
          active: true,
        },
      ]}
    ></Breadcrumb>
  </Story>
</Preview>

### ARIA example with subcomponents

<Preview>
  <Story name="Breadcrumb/ARIASubcomponent">
    <Breadcrumb>
      <BreadcrumbItem href="../..">WAI-ARIA Authoring Practices 1.1</BreadcrumbItem>

      <BreadcrumbItem href="../../#breadcrumb">Breadcrumb Patterns</BreadcrumbItem>

</Preview>
